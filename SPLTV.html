Persamaan Awal (zahra)

1. x + y + z = 125.000

2. x + 2y + z = 165.000

3. 2x + 2y + z = 200.000

Nilai Awal

X = 35.000

Y = 40.000

Z = 50.000

Eliminasi z antara Persamaan 2 dan 1

Persamaan 2: x + 2y + z = 165.000

Persamaan 1: x + y + z = 125.000

Hasil eliminasi:

y = 40.000

Eliminasi z antara Persamaan 3 dan 2

Persamaan 3: 2x + 2y + z = 200.000

Persamaan 2: x + 2y + z = 165.000

Hasil eliminasi:

x = 35.000

Substitusi x dan y ke Persamaan 1

Substitusi x = 35.000 dan y = 40.000

35.000 + 40.000 + z = 125.000

75.000 + z = 125.000

z = 125.000 - 75.000

z = 50.000

Hasil Akhir (HP)

HP = {(x, y, z)}

HP = {(35.000, 40.000, 50.000)}

Sistem Persamaan Linear (gilang)

1. x + 2y + 2z = 180.000
2. 2x + 2y + z = 200.000
3. 2x + y + 2z = 195.000

x → tempat pensil
y → motherboard
z → mug

Eliminasi x – 1 dan 2

x + 2y + 2z = 180.000 |.2
2x + 2y + z = 200.000 |.1
↓
2x + 4y + 4z = 360.000
2x + 2y + z = 200.000
-----------------------
2y + 3z = 160 (Persamaan 4)

Eliminasi x – 2 dan 3

2x + 2y + z = 200.000
2x + y + 2z = 195.000
-----------------------
y + (-z) = 5.000 (Persamaan 5)

Eliminasi y – 4 dan 5

2y + 3z = 160.000
y + (-z) = 5.000
-----------------------
5z = 15.000
z = 150.000
z = 30.000

Substitusi z ke 4

2y + 3z = 160.000
2y + 3.30.000 = 160.000
2y + 90.000 = 160.000
2y = 160.000 - 90.000
2y = 70.000
y = 70.000 : 2
y = 35.000

Substitusi y dan z ke 1

x + 2y + 2z = 180.000
x + 2.35.000 + 2.30.000 = 180.000
x + 70.000 + 60.000 = 180.000
x + 130.000 = 180.000
x = 180.000 - 130.000
x = 50.000

Hasil

HP = {(x, y, z)}
HP = {(50.000, 35.000, 30.000)}

Sistem Persamaan Linear (khalisha)


X = harga 1 meja Ecobrick
Y = harga 1 tempah sampah dari tutup botol
Z = harga 1 bunga dari plastik

X + Y + Z = 120.000 ①
2X + Y + 2Z = 205.000 ②
3X + 3Y + 2Z = 385.000 ③

2 × (X + Y + Z = 120) → 2X + 2Y + 2Z = 240
2X + Y + 2Z = 205
─────────────
Y = 35

3X + 3Y + 2Z = 385
3X + Y + 2Z = 205
─────────────
2Y = 180 → Y = 90 (Tapi sudah diketahui Y=35, ini pembuktian)

Substitusi Y=35 ke persamaan pertama:
X + 35 + Z = 120
X + Z = 85

Substitusi Y=35 ke persamaan kedua:
2X + 35 + 2Z = 205
2X + 2Z = 170

Dari X + Z = 85 → Z = 85 - X
Substitusi ke 2X + 2Z = 170:
2X + 2(85 - X) = 170
2X + 170 - 2X = 170
170 = 170 (benar)

Nilai:
X = 60.000,
Y = 35.000,
Z = 25.000
